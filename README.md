# Loop
## Project Abstract

A LOOP é um marketplace de dispositivos eletrónicos recondicionados que coloca a transparência e a sustentabilidade no centro de cada transação. Ao contrário das alternativas existentes no mercado, a LOOP disponibiliza o histórico completo de intervenções e reparações de cada dispositivo, para que o comprador saiba exatamente o que está a adquirir. Cada produto vem acompanhado de uma garantia de 24 meses, com possibilidade de extensão até 36 meses, oferecendo uma segurança que vai muito além do que é habitual neste tipo de plataformas. Para além disso, a LOOP mostra o impacto ambiental real de cada compra, tornando cada transação num ato de consumo consciente.
